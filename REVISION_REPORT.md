○ 신규 컴포넌트
- **ResetButton.jsx** — 헤더 우측에 "↺ 리셋" 회색 버튼
- 입력값 / 보기모드 / 시도 / 시군구 / 색상 설정 모두 초기값 복원
- 변경사항 있을 때만 confirm 다이얼로그 표시 (실수 방지)

○ 엑셀 업로드 강건성 강화
| 검증 항목 | 처리 |
|----------|------|
| .xlsx/.xls 외 확장자 | "xlsx 또는 xls 파일만 지원함" 에러 |
| 50MB 초과 | "파일 크기 50MB 초과" 에러 |
| 빈 파일 (시트 없음) | "시트가 없는 빈 파일" 에러 |
| 데이터 행 0건 | "데이터 행이 없음" 에러 |
| 지역코드 컬럼 미발견 | "지역코드 컬럼을 찾지 못함. 양식 다운로드 후 재시도" |
| 값 컬럼 미발견 | "값 컬럼을 찾지 못함. 양식 다운로드 후 재시도" |
| 매칭 0건 | "매칭된 데이터 0건. 양식이 잘못되었거나 보기 모드가 맞지 않음" |
| 매칭 실패율 50%↑ | 노란색 경고 박스 출현 ("보기 모드 또는 양식 파일 재확인") |
| 입력양식 외 시트 사용 | 노란색 안내 ("'{시트명}' 시트 사용됨") |

○ 코드 매칭 강화
- 앞자리 0 손실 자동 보정 (4→5자리, 7→8자리, 9→10자리 패딩)
- 행안부 10자리(adm_cd2) → 통계청 8자리(adm_cd) 자동 변환

○ 에러 발생 시 즉시 리셋 버튼
- 에러 박스 안에 "입력값 초기화 후 재시도" 버튼 동봉
