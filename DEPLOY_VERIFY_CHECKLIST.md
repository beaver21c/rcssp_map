## 7. 트러블슈팅 (빈도순)

### □ 7-1. Actions 빨간 X (Pages 활성화 단계)
- 원인: enablement 옵션 잘못 적용
- 처방: deploy.yml 다시 점검 → `enablement: true` 위치 확인 (configure-pages@v5의 with: 하위)

### □ 7-2. 빌드 단계에서 npm ci 실패
- 원인: package-lock.json 불일치
- 처방: 로컬에서 `npm install` 재실행 → package-lock.json commit

### □ 7-3. 빌드 성공이지만 URL 404
- 원인: vite.config.js base 경로 불일치
- 처방: `GH_PAGES=1` 환경변수 시 base가 `/rcssp_map/` 인지 확인

### □ 7-4. 페이지 로드는 되지만 데이터 미표시
- 원인: TopoJSON 경로 오류
- 처방: 브라우저 콘솔(F12) → Network 탭 → 404 자원 확인

### □ 7-5. 한글 깨짐
- 원인: charset 미설정 또는 BOM 문제
- 처방: `<meta charset="utf-8">` 확인 + 빌드 결과물 인코딩 확인

### □ 7-6. PNG 다운로드 안 됨
- 원인: html2canvas 로딩 실패 또는 CORS 차단
- 처방: 콘솔 에러 메시지 확인 + 클린 모드 토글로 재시도
